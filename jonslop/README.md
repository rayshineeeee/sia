# jonslop

Ideation and documentation workspace for what we build on top of [`sia`](../).

This folder is deliberately loose — a scratchpad for thinking out loud, capturing
ideas, sketching directions, and writing up findings **before** committing to what
and how we build. Nothing here is final until we promote it.

## Suggested layout (adapt freely)

- `ideas/` — raw concepts, one file per idea, no quality bar
- `research/` — notes on the `sia` codebase, prior art, constraints
- `decisions/` — write-ups of choices we land on and why
- `scratch/` — anything transient
